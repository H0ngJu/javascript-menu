# 기능 구현

1. 코치 이름을 입력 받는다.
   1. 코치 정보 만들기
      1. 이름은 2이상 4 이하
      2. 코치는 2명 이상
2. 코치 이름에 따라 못먹는 메뉴를 입력한다.
   1. 입력한 정보를 코치 정보에 dont 에 넣는다.
   2. 메뉴가 2개 이하여야 한다.
3. 일주일동안의 카테고리를 선정한다.
   1. 2번 나왔던 카테고리는 다시 뽑는다.
4. 선택된 요일에 대하여, 각 코치 별로 아래를 수행한다.
   1. 못먹는 메뉴 (dont)에 속해있으면 다시 랜덤을 돌린다.
   2. 이미 먹었던 메뉴 (recommend)에 들어있으면 다시 랜덤을 돌린다.
